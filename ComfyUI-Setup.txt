HOW TO GENERATE INSIDE COMFYUI  /  COMO GENERAR EN COMFYUI

[EN] Open "Template-Composer.html" in your browser. Pick a template, fill the
fields (or roll the dice), then use the ComfyUI box: set the address
(default http://127.0.0.1:8188), pick an orientation and click Send.
To allow the Send button, start ComfyUI with CORS enabled:
    python main.py --enable-cors-header
You can also drag any image from this pack onto the ComfyUI canvas to load its workflow.

[ES] Abre "Template-Composer.html" en tu navegador. Elige un template, rellena los
campos (o usa el dado) y en el bloque ComfyUI pon la direccion
(por defecto http://127.0.0.1:8188), elige orientacion y pulsa Send.
Para que funcione el boton Send, arranca ComfyUI con CORS activado:
    python main.py --enable-cors-header
Tambien puedes arrastrar cualquier imagen del pack al lienzo de ComfyUI para cargar su workflow.

[KO] 브라우저에서 "Template-Composer.html"을 엽니다. 템플릿을 선택하고 필드를 채운 뒤,
ComfyUI 칸에 주소(기본값 http://127.0.0.1:8188)를 입력하고 방향을 선택한 후 Send를 누르세요.
Send 버튼을 사용하려면 CORS를 활성화하여 ComfyUI를 실행하세요:
    python main.py --enable-cors-header
이 팩의 이미지를 ComfyUI 캔버스로 드래그하면 워크플로우가 로드됩니다.

[ZH] 在浏览器中打开 "Template-Composer.html"。选择模板并填写字段，然后在 ComfyUI 区域
输入地址（默认 http://127.0.0.1:8188），选择方向并点击 Send。
要使用 Send 按钮，请在启用 CORS 的情况下启动 ComfyUI：
    python main.py --enable-cors-header
您也可以将本包中的任意图片拖到 ComfyUI 画布上以加载其工作流。
